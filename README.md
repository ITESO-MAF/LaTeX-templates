
# LaTeX-templates

A collection of templated-projects in LaTeX for academic publications

## ITESO-Thesis

- [**ITESO_Thesis**](ITESO_Thesis/) Contains a format often used for a graduate thesis document, and/or, a thesis-like document in LaTeX.
